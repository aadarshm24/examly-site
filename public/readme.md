# 📚 Examly — Your smart academic companion.
A minimal, aesthetic, and fast learning platform built for engineering students.  
Get **Notes, PYQs, and Question Banks** instantly — organized by semester.

---

## ✨ Live Demo  
👉 https://examly-site.vercel.app/

---

## 🎯 Features  
- 🌓 **Dynamic Dark & Light Mode** — toggle for day or night study sessions.
- 📚 **Semester-Based Workflow** — Select your semester to load tailored curriculum.
- ⚡ **Instant Resource Access** — Direct Google Drive links for **Notes, PYQs, and Q-Banks**.
- 💬 **Daily Motivation** — Welcome screen with randomized inspirational quotes.
- 🎨 **Premium UI/UX** — Glassmorphism, smooth GSAP animations, and floating 3D elements.
- 📱 **Fully Responsive** — Optimized for mobile, tablet, and desktop.

---

## 🛠 Tech Stack  
- **Core:** HTML5, Modern JavaScript (ES6+)
- **Styling:** Tailwind CSS + Custom CSS Variables
- **Animations:** GSAP (GreenSock Animation Platform)
- **Icons:** Lucide Icons
- **Deployment:** Vercel

---

## 📂 Project Structure

```bash
public/
│── index.html        # Home: Welcome screen, Quotes, Semester Modal
│── semester.html     # Curriculum: Subject grid & Resource links
│── contact.html      # Contact form & Developer info
│── style.css         # Global styles & Dark/Light mode logic
│── script.js         # Core logic, Data structure, GSAP animations
│── Study.png         # 3D Hero Image
│── favicon.png       # Site Favicon
└── vercel.json       # Deployment config
```
